# Módulo `validador_duplicados`

Este módulo analiza archivos Markdown entre los cuadernos temáticos y la bitácora central de ALMA LIBRE para detectar duplicados mediante hash y similitud textual.

## Características

- Comparación por hash SHA-256
- Detección de similitud textual (>90%)
- Validación de metadatos YAML
- Reporte estructurado en YAML

## Uso

```bash
python3 validador_duplicados.py
python3 validador_duplicados.py --umbral 0.85 --origen CUADERNOS/trading.cu/docs/
```

## Salida

Se genera un archivo `.yaml` en:

```
MODULOS/validador_duplicados/output/reporte_sugerencias_YYYYMMDD_HHMM.yaml
```
